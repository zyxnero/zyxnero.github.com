关闭 HTC 8x 按键震动及按键灯

Update

  • 2014.05.24 经长期试用，发现关闭电容键震动后，电话拨通的震动提示以及其它震动提示也都没了，权衡之后恢复按键震动。由此：关闭按键灯以及电容键震动均以失败告终，默认就好。
  • 2014.02.07 按键灯关闭貌似是临时的，几个小时就恢复了

WP 果然是没有 Android 开放，耗电速度也让我和我的小伙伴惊呆了，所以毅然决定关闭按键震动及按键灯让爪机更省电，方法记录如下。

关闭按键震动

手机拨号界面 → 输入##634# → Accept → Vibrator Test → Run → Off → 直接退出

关闭按键灯

手机拨号界面 → 输入##634# → Accept → Led Test → Run → Key LED → 直接退出

注：重启恢复
