在网页中嵌入 hermit 播放器

Mar 08,2014

Hermit 播放器是牧风兄整的一个 WordPress 音乐播放器插件，直接通过虾米的歌曲、专辑或精选集 ID 引用虾米的歌曲链接实现播放，且播放器的 css 写成自适应样式，在移动设备上的表现良好。实现方式应该是写了 API，具体的我也不太懂，可以直接去插件发布页查看。
