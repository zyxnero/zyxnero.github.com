置 footer 于页面底部

前几天发现有些内容比较少的文章页面，其 footer 会上移浮在半空，看起来着实别扭，所以又调整了一下 CSS。

由于现主题是基于 bootstrap 的，在 Bootstrap 的 example 中找到了 2 份示例：Bootstrap2 示例，Bootstrap3 示例。我选择模仿套用 Bootstrap2 的示例（感觉简单一点），在主题的 CSS 中添加了下面 2 行：

html,body {height: 97.7%;}
#body-content{min-height:100%;height:auto !important;}

至于 Bootstrap3 的示例中采用的 position: absolute;bottom: 0; 的好处，或者说 Bootstrap2 的示例中采取的办法的缺陷，还不清楚，以后发现了再改……
