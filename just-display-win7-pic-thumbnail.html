Win7 桌面如何只显示图片文件缩略图

Win7 默认开启桌面缩略图功能时图片文件、pdf、ppt、视频文件的缩略图都会在桌面显示出来，以下三步可开启图片文件的桌面缩略图而不显示其余类型的缩略图。

阉割掉 pdf、ppt、视频文件的缩略图功能。

具体操作：进注册表，分别找到需要修改的键值，将其 shellex 键给删掉。注册表文件内容如下 

Windows Registry Editor Version 5.00
[-HKEY_CLASSES_ROOT.avi\ShellEx]
[-HKEY_CLASSES_ROOT.asf\ShellEx]
[-HKEY_CLASSES_ROOT.wmv\ShellEx]
[-HKEY_CLASSES_ROOT.rm\ShellEx]
[-HKEY_CLASSES_ROOT.rmvb\ShellEx]
[-HKEY_CLASSES_ROOT.mkv\ShellEx]
[-HKEY_CLASSES_ROOT.3gp\ShellEx]
[-HKEY_CLASSES_ROOT.3g2\ShellEx]
[-HKEY_CLASSES_ROOT.3gp2\ShellEx]
[-HKEY_CLASSES_ROOT.3gpp\ShellEx]
[-HKEY_CLASSES_ROOT.f4v\ShellEx]
[-HKEY_CLASSES_ROOT.flv\ShellEx]
[-HKEY_CLASSES_ROOT.m4v\ShellEx]
[-HKEY_CLASSES_ROOT.mov\ShellEx]
[-HKEY_CLASSES_ROOT.mp4\ShellEx]
[-HKEY_CLASSES_ROOT.mp4v\ShellEx]
[-HKEY_CLASSES_ROOT.mpg\ShellEx]
[-HKEY_CLASSES_ROOT.mpeg\ShellEx]
[-HKEY_CLASSES_ROOT.3gp\ShellEx]
[-HKEY_CLASSES_ROOT.3g2\ShellEx]
[-HKEY_CLASSES_ROOT.3gp2\ShellEx]
[-HKEY_CLASSES_ROOT.3gpp\ShellEx]
[-HKEY_CLASSES_ROOT.f4v\ShellEx]
[-HKEY_CLASSES_ROOT.flv\ShellEx]
[-HKEY_CLASSES_ROOT.pdf\ShellEx]
[-HKEY_CLASSES_ROOT.pptx\ShellEx]
[-HKEY_CLASSES_ROOT\AcroExch.Document.7\Shellex]
[-HKEY_CLASSES_ROOT\SystemFileAssociations.ppt\shellex]
[-HKEY_CLASSES_ROOT\SystemFileAssociations.mpp\shellex]
[-HKEY_CLASSES_ROOT\SystemFileAssociations.msg\shellex]
[-HKEY_CLASSES_ROOT\SystemFileAssociations.ppt\shellex]

关掉缩略图的数据库

进组策略：Win + R → gpedit.msc → 用户配置 → 管理模板 → Windows 组件 → Windows 资源管理器，双击右边的“关闭隐藏的 thumbs.db 文件中的缩略图片缓存” → 选择“已启用”。

删除之前已经存档的缩略图

即删除缩略图的数据库 thumbs.db 文件，它们都在系统盘的 Users\（当前用户）\AppData\Local\Microsoft\Windows\Explorer 下，用 Diskgenius 或在另一系统下将其删除，之后重启。开启缩略图功能，这次显示缩略图的就只有图片文件了。
