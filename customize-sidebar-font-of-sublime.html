Sublime 侧边栏字体大小调整

Apr 23,2017

最近感觉视力好像又下降了，无奈。前几天调大了 Sublime Text 3 的侧边栏字体使正文区和侧边栏看起来更和谐。过程记录如下。

安装插件 PackageResourceViewer

  1. 输入 Ctrl + Shift + P 调出命令面板；
  2. 输入 Package Control:Install 回车，等待加载 package 列表；
  3. 搜索并安装 PackageResourceViewer 包。

修改侧边栏字体

  1. 输入 Ctrl + Shift + P 调出命令面板；
  2. 输入 PackageResourceViewer: Open Resource 进入到 XXX - Default 主题的目录下，打开对应的 XXX.sublime-theme 主题文件。（XXX 为待修改主题的名称）
  3. 搜索 sidebar_label 关键词，在 "class": "sidebar_label" 后面增加一行："font.size": 13,;
  4. 若要更改标签栏的字体和大小，同样是在该文件中搜索 tab_label 关键词，在 "class": "tab_label" 后增加增加一行："font.size": 12,。

更改后，我的侧边栏字体设置如下：

"class": "sidebar_label",
"font.size": 14,
"color": [180, 180, 180],
"shadow_color": [0, 0, 0],
"shadow_offset": [0, 0.5]

标签栏字体设置如下：

"class": "tab_label",
"font.size": 12, 
"fg": [180, 180, 180],
"shadow_color": [0, 0, 0, 200],
"shadow_offset": [0, 0.5]
