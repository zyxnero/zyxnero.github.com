Mason 两个基础用法

Feb 27,2015

Mason 是国人 tiramn 写的一个强大扩展。目前已实现功能如下：

  1. 同步下载网页音频和视频；
  2. 重定向 url / 阻止请求；
  3. 支持脚本，类似于 Grease Monkey，区别在于执行效率更高，使用更加灵活；
  4. 修改 http header。

以下内容系网上搜集，并根据个人实际应用堆叠……

反防盗链

用途为：让你能看到发图人以为别人也能看到的图片，如秋秋空间，百度图片，新浪博客上的相册。它们在被引用页不属于自己的网页时就会显示不正常。如 RSS Reader 里面就会出现一些图片无法显示。Mason 默认规则已解决上述几个站点的防盗链问题。下面沿用卡饭论坛上的例子（破图在 chrome 下真的很难看）。

Mason 反防盗链设置

安装 Mason 后添加规则，如图设置，① 为防盗链图片包含网址，② 为该图能正常显示的网址，一般情况下 ① 与 ② 是同一个网址。

update 2015.02.28：大发博客 及 zdfans 的站在 Inoreader 中均无法显示图片，经如上设置即可解决。

Mason 重定向

使用联通网时 Google 搜索结果中的 wikipedia 条目打开后会直接跳转到 http://nfdnserror3.wo.com.cn。将 http://zh.wikipedia.org 重定向到 https://zh.wikipedia.org 或 http://zh.wikipedia.org/zh-cn/ 即可解决问题。

Mason 重定向设置
