Lusir Lubuntu - 软件篇

安装完 Lubuntu，桌面上啥都没有，还真不知从何下手。不过不能下手也得下手，一步一步来。

换源

新立得软件包管理器 → 设置 → 软件库 → 改“下载自”为 163.com 源。

安装完整的语言支持

首选项 → 语言支持

修改 hosts

系统默认编辑器为 leafpad 编辑器，hosts 文件在 etc 目录下，所以命令为

sudo leafpad /etc/hosts

打开 smartladder 站点，将更新的 host 列表复制到刚刚打开的编辑器中保存即可。

安装输入法

在软件包管理器中搜寻 fcitx，安装 fcitx 与 fcitx googlepinyin

安装 Flash 插件

我没有用命令行，是在软件包管理器中寻找安装的。

安装 QQ

可以使用网上推荐的 Webqq，或 pidgin-lwqq + pidgin，命令如下：

sudo add-apt-repository ppa:lainme/pidgin-lwqq
sudo apt-get update
sudo apt-get install pidgin-lwqq

然后打开 pidgin，添加 WebQQ 协议的账号即可。

直接在在软件包管理器中搜寻 Catfish 安装即可，添加到右键菜单参考这一篇
