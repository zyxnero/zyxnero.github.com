Markdown 添加表格

搭建了 Pelican 之后就在慢慢尝试学习使用 Markdown 记录一些东西（包括尝试使用 Markdown 编辑邮件）。本文记录一下使用 Markdown 添加表格的方法。

简单的表格

你好 | Markdown | 表格 | 这是表头
---- | -------- | ---- | --------
abc  |  123   |  ABC  |  cool
111  |  222   |  333  |  444

效果

你好 Markdown 表格 这是表头
abc 123 ABC cool
111 222 333 444

对齐的表格

|Tables     |Are             |Cool |
| ----------- |:-------------:| -----:|
| col 3 is    | right-aligned | $1600 |
| col 2 is    | centered      |   $12 |
| zebra stripes | are neat      |    $1 |

效果

Tables Are Cool
col 3 is right-aligned $1600
col 2 is centered $12
zebra stripes are neat $1
