尝试使用 TortoiseGit

每次使用 git 上传目录都要敲代码，对于我这还是太过艰难，于是装了个 TortoiseGit 有了这货，目录更新之后只用右键提交，方便不少，当然前提是 TortoiseGit 要设置正确

下面记录2点以作备忘：
1. 设置 git 地址时要使用 git@github.com:×××/××.git ，而不要使用 https 开头的地址
2. 让 TortoiseGit 记住某个 repo 的账号密码（明文记录）：进入 “repo/.git/config” ，增加 

[credential]
    helper = store
