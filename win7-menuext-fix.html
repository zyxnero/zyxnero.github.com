去除 Win7 右键“使用 Windows Media Player 播放”项

Nov 09,2014

我不是处女座，但还是会对右键菜单非常在意，经常会留意右键菜单是否需要清理，经常使用的工具为：RightMenuMgr。今天记录一下 .avi 文件右键菜单上“使用 Windows Media Player 播放”项的整理过程。

如下图，由于我常不使用 Windows Media Player，所以该菜单项对于我来讲很多余。

其实，这主要是 Windows 7 的默认设置在发生作用。解决的办法很简单：

点击开始菜单中的“默认程序”→“设置程序访问和计算机的默认值”。

在弹出的窗口中选择“自定义”并单击右边的向下箭头。然后取消对“选择默认的媒体播放机”下的 “Windows Media Player” 与 “Windows Media Center” 右边的“启用对此程序的访问”的勾选。

至此，媒体文件的右键菜单将不再出现与 Windows Media Player 有关的多余选项了。
