为 TortoiseGit 设置代理

又一篇流水账。昨天“中菲南海仲裁案”出结果，今天就发现博客内容不能更新到 Github 了。老毛病，为了提交更新，只能为 TortoiseGit “另辟蹊径”了。

  1. 打开“袋里”并启用系统袋里；
  2. 留意袋里程序提供的端口，举个栗子：54213；
  3. 打开 TortoiseGit→进入设置→打开全局设置文件（如下图）

  1. 在全局设置文件中添加以下两行
[http]
    proxy = http://127.0.0.1:54213
    sslVerify = false

再次推送，成功提交！！！
