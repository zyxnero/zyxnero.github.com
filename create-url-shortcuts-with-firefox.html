利用 Firefox 为网页链接创建桌面快捷方式

精简掉桌面右键的 新建快捷方式 后，发现原来还可以通过 Firefox 为网页链接创建桌面快捷方式：

  1. 调整 Firefox 窗口，直到可以在同一个屏幕里同时看到桌面和 Firefox；
  2. 按住 地址栏 左边的图标；
  3. 按住鼠标左键的同时，将指针移动到桌面，然后松开鼠标左键，快捷方式就创建好了。

其实，如果是在 Win7 以上版本的 Windows 下，可以直接将图标拖至屏幕右下角，待返回至桌面松开鼠标左键即可创建。

同理，我们也可以拖放书签菜单、书签和历史记录栏或者 我的足迹 窗口中的项目到桌面上从而为其创建快捷方式。
