用上了 You-Get

Jul 14,2016

在优酷上订阅了几个视频节目，像《罗辑思维》、《暴走大事件》。之前一直在用硕鼠下载观看，不过前阵子硕鼠好像停止服务了，遂用上了开源跨平台的 You-Get。之前在 Lubuntu 下使用过，绝对的神器！

从官网下载压缩包，解压即用。需要注意的是如果命令行里面提示“无法定位程序输入点ucrtbase.terminate”这个错误，那么去下载个Microsoft Visual C++ 2015 Redistributable，安装以后就不会再报错了。如果命令行程序用的是 Cmder，那么看起来就是这样的。

updated 2017.04.18：近期 You-Get 的 Release 不提供 Win 下的预编译文件了，所以我使用 pip install --upgrade you-get 命令在 Python3 的 Scripts 目录下生成了 you-get.exe 并将它扔到了 Python2 的 Scripts 目录下（博客文章渲染需要使用 Python2），这个 100k 不到的 exe 文件居然也能正常下载合并文件，真神奇！

updated 2018.07.12：最近用 You-Get 下载 B 站视频时，发现长视频默认仅下载第一部分，下载全部需要使用 -l 命令，如下：

you-get -l https://www.bilibili.com/video/av26508282

分段下载完成后可以参照 这一篇记录 进行合并。
