Pelican + Niux2 使用记录

Update

  • 2014.01.25 将静态目录由 gitcafe 转移至 github，搜索栏高度改回 Niux2 主题默认。
  • 2014.02.11 添加 html5media.min.js 至主题 base.html 中。
  • 2014.02.17 给 index、article、page及 404 页面添加 meta。

果然还是需要一个可以宣泄情感垃圾的地方。域名被恶意泛解析之后就没管了，闲置大半年。最近看网上教程学着搭了个静态博客，接触到了 git、python、pelican 等名词，很新鲜。于是又动了记录一些生活琐事的念头。

现在使用的是 Mawenbao 的 Niu-X2 主题，很好看，很喜欢，希望以后能更加自适应。记录一下启用主题之后的微小修改：

  1. 改主题下所有 css 文件中 underline 为 none；
  2. 修改 footer；
  3. 强制显示浏览器侧边栏（niu2.css）； 
    body, html {min-height:101%;}
  4. 文章侧边栏增加 Author，显示 fa-bug 图标；
  5. 修改搜索栏高度（Chrome 跟 Firefox 下显示高度不一样）。

Windows 下使用 python 貌似还是有些问题，路径的写法跟网上找到的不一样，导致 extra 目录下的文件是不能拷到根目录。
