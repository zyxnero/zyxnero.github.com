使用七牛镜像存储

看到很多博主都正在使用七牛作为图床或整站的 CDN 加速，本着外行凑热闹的精神，我也学着使用了一把。七牛的加速节点响应速度确实是很快，至少目前是这样，我 ping 了一下，结果如下。

下面记录一下七牛镜像存储的使用（本例为图床使用）：

由于我没有备案的打算，所以使用的是七牛提供的三级域名。

假设我想要镜像的源站为 img.lusir.me。而且我已经建立了名为 pic 的七牛 bucket（bucket 意指“空间”，理解为“坑”我觉得更形象）。

  1. 进入七牛内容管理页面，设置一键加速网站，修改镜像源为 img.lusir.me，底下的“镜像空间”默认是 pic（如果我只建立了一个“坑”）。这时点击“确定加速”即可将 img.lusir.me 与 pic.u.qiniudn.com 进行关联。

  2. 更改文章里面的图片地址，搜寻关键字 img.lusir.me 并将其改为 pic.u.qiniudn.com，保存。

  3. 访问一篇带图的文章，进入七牛内容管理页面的内容管理，此时文章里相应的图会被拉取到七牛的“坑”里面。也可以直接将所有的图通过QRSBox上传到“坑”里面。

如此就完成整个迁移过程。此时 img.lusir.me 这个源站就可以暂时废弃不用了，即以后可以直接将 pic.u.qiniudn.com 作为图床。

如果要保留源站的域名方式 img.lusir.me，七牛也提供了方法，点我查看
