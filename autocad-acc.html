优化 AutoCAD 启动运行速度

Dec 11,2014

最近几个版本的 AutoCAD 对于系统资源的要求越来越大了，还是老版本轻便一些。采用以下几个步骤将 AutoCAD 不必要的加载除掉。

步骤一

安装 AutoCAD 产品后，顺便安装 AutoCAD 管理工具，亦即 CMControl。安装完成后打开 CAD 管理员控制实用程序，选择要修改的产品，即 CAD 版本号，禁用所有通信。

步骤二

运行 AutoCAD 后，打开进程管理器，会发现这样一个进程：WSCommCntr1.exe，它是 AutoCAD 的后台程序，会发送 AutoCAD 的版本信息和注册信息给 AutoDesk 公司，并自动收集相关信资料，包括图集、信息、更新等。

将 C:\Program Files (x86)\Common Files\Autodesk Shared 目录下的 WSCommCntr1.exe 与 AutoCAD 安装目录下的 WSCommCntrAcCon.arx 重命名。

步骤三

在 AutoCAD 命令行中输入 VTENABLE，依照提示输入 0 以启用平滑切换显示区域，加快启动速度，接着回车。

步骤四（待验证）

打开 IE 浏览器，将“Internet 选项”→“高级”选项卡下的“检查发行商的证书是否已吊销”项取消勾选。

步骤五

这一步是禁止 AutoCAD 联网，ESET Smart Security 及众多防火墙都能实现，参考ESET Smart Security 添加黑白名单。


近日 Autodesk 公司发布了 AutoCAD 2015，同时宣布向全球学生教师和教育机构提供全功能教育版，包括 AutoCAD、3DMax、Maya 等。不管出于什么目的，不管其影响如何，这种行为还是值得点个赞的！
