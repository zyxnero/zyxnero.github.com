升级到 Sublime Text 3

Jul 02,2016

常用 Sublime Text 2 和 Notepad++ 编辑文本，非重度使用。忍不了 Sublime Text 2 启动速度的时候就换用 Notepad++，也还将就。Sublime Text 3 出现的时候也试用过，当时插件不够给力，也没太在意启动速度，遂弃之。前几天心血来潮又试了一下 Sublime Text 3，于是这一次从右键菜单里消失的是 Sublime Text 2 和 Notepad++。

  1. 从官方下载地址下载对应的 Portable 版本并解压到自定义目录；

汉化

我用的好像是 荒野无灯 提供的 Sublime Text 2 汉化包，扔到 Sublime Text 3 里发现可以通用，将汉化包文件（Default.sublime-package）拷到程序目录下的 \Data\Installed Packages 中，无需重启即可看到汉化效果。如果不想看到菜单栏“帮助”下汉化者的太多个人信息，可以编辑 \Data\Packages\Default 目录下的 Main.sublime-menu 文件，搜索汉化者的个人信息关键字，将其注释掉。

插件

首先是要装上 Package Control 组件，以后就利用它来管理各种插件。

通过 Ctrl + ` 打开调试窗口，在输入框内粘贴如下代码，然后回车自动安装，安装完成重启。

import urllib.request,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)
